# B103 Hotel Reservation System

This project is a relational database system created for the B103 Databases & Big Data module.

## Files
- `hotel_reservation_system.sql`: SQL schema, sample data, and example queries
- `er_diagram_hotel.png`: ER diagram of the database
- `B103_Hotel_Reservation_Report.pdf`: Final report

## Domain
Hotel Reservation System

## Features
- Customer management
- Room management
- Booking records
- Payment tracking
- SQL joins and aggregation queries
